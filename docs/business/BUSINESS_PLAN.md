## 1. 사업 개요

### 1.1 비전

> "말 한마디로 예약 완료" — 대한민국 최초 AI 파크골프 통합 플랫폼

### 1.2 사업 방향

| 단계 | 목표 | 핵심 전략 |
|------|------|----------|
| **현재** | 초기 고객 확보 | AI 부킹으로 편의성 극대화, 무료 가맹점 유치 |
| **1단계** | 수익 기반 구축 | 예약 수수료 모델 도입 |
| **2단계** | 사업 확장 | 레슨 마켓플레이스 "메이트 레슨" |

### 1.3 사업 구성

```
파크골프메이트
├── 사용자 앱 (iOS / Android / Web)
│   ├── AI 자연어 예약
│   ├── 그룹 채팅방 단체 예약
│   ├── 더치페이 분할결제
│   └── 소셜 (친구, 채팅, 모임)
│
├── 가맹점 관리 대시보드 (Web)
│   ├── 예약/슬롯 관리
│   ├── 매출 분석
│   ├── 고객/노쇼 관리
│   └── 정책 설정 (취소/환불/노쇼/운영)
│
└── 플랫폼 관리 대시보드 (Web)
    ├── 전국 가맹점 통합 관리
    ├── 통합 통계 및 분석
    ├── 계층형 정책 관리 (PLATFORM → COMPANY → CLUB)
    └── 관리자/역할/권한 관리 (RBAC)
```

## 2. 시장 현황과 기회

### 2.1 파크골프 시장 규모

| 지표 | 수치 | 비고 |
|------|------|------|
| 이용자 수 | **150만+** | 2024년 기준 |
| 전국 파크골프장 | **490+** | 운영 중 |
| 10년 성장률 | **853%** | 2013~2023 |
| 연간 성장률 | **30%+** | 최근 5년 평균 |

### 2.2 이용자 특성

| 항목 | 내용 |
|------|------|
| 핵심 이용층 | 50~70대 시니어 |
| 성별 비율 | 남성 71.5%, 여성 28.5% (여성 참여율 증가 추세) |
| 이용 동기 | 건강 관리, 사교 활동, 여가 |
| 이용 빈도 | 주 2~3회 (동호회 기반 정기 라운드) |
| 1회 이용료 | 5,000~15,000원 |

### 2.3 시장의 문제점

| 문제 | 현황 | 영향 |
|------|------|------|
| **디지털 인프라 부재** | 150만 이용자가 전화/수기 예약 | 시니어 불편, 운영 비효율 |
| **전화 예약** | 6단계, 3~5분 소요, 영업시간만 가능 | 예약 포기, 노쇼 관리 불가 |
| **기존 앱** | 9단계 복잡 UI, 시니어 진입장벽 높음 | "그냥 전화가 편해" |
| **레슨 플랫폼 부재** | 파크골프 전문 레슨 매칭 서비스 없음 | 레슨 수요가 비공식 채널에 분산 |
| **가맹점 관리** | 수기 장부, 노쇼 관리 불가, 통계 부재 | 운영 비효율, 매출 분석 불가 |

### 2.4 기회 요약

```
급성장하는 시장 (연 30%+) + 디지털 인프라 전무 + 레슨 플랫폼 부재
= 선점 기회
```

## 3. 서비스 소개

### 3.1 3대 핵심 가치 — 간편 부킹 · 간편 결제 · 더치페이

파크골프메이트는 시니어 이용자의 가장 큰 불편 3가지를 해결합니다.

**간편 부킹 — "말 한마디로 예약 완료"**

| 비교 | 전화 예약 | 기존 앱 | 파크골프메이트 |
|------|---------|--------|-------------|
| 예약 단계 | 6단계 | 9단계 | **1단계** |
| 소요 시간 | 3~5분 | 2~3분 | **~15초** |
| 가능 시간 | 영업시간만 | 24시간 | **24시간** |
| 빈 시간 검색 | 불가 | 수동 | **자동** |
| 날씨 확인 | 불가 | 불가 | **자동** |
| 단체 예약 | 개별 연락 | 개별 예약 | **@AI 한번으로** |

```
사용자: "천안 골프장 2명 예약해 줘"
  → AI: 골프장 검색 + 날씨 확인 + 빈 슬롯 조회
  → 사용자: 슬롯 선택 + 결제방법 선택
  → AI: 예약 완료 (~15초)
```

**간편 결제 — "터치 한 번으로 결제 완료"**

| 비교 | 현장 결제 | 기존 앱 | 파크골프메이트 |
|------|---------|--------|-------------|
| 결제 방식 | 현장 방문 후 결제 | 복잡한 결제 UI | **Toss 간편결제** |
| 결제 수단 | 현금/카드 | 카드 | **카드 + 간편결제 + 더치페이** |
| 수수료 | 없음 | 불명확 | **투명한 5%** (현장결제는 무료) |
| 예약 보장 | 없음 (노쇼 가능) | 일부 | **결제 즉시 확정** |
| 환불 | 전화 요청 | 복잡한 절차 | **자동 환불 처리** |

**더치페이 — "각자 결제, 정산 고민 끝"**

파크골프는 동호회·모임 중심의 단체 활동입니다. 모임 후 정산은 매번 반복되는 불편입니다.

```
동호회 정기 라운드 (4인)
  → 예약자가 "더치페이" 선택
  → 참여자 4명에게 개별 결제 링크 자동 발송 (push + 채팅)
  → 각자 터치 한 번으로 결제 완료
  → 예약자: 실시간 정산 현황 확인
  → 정산 완료 → 예약 확정
```

| 비교 | 현재 정산 방식 | 파크골프메이트 더치페이 |
|------|-------------|-------------------|
| 방식 | 1인 대표 결제 → 카카오페이 송금 요청 | **각자 개별 결제** |
| 문제 | 미정산, 독촉, 금액 착오 | **자동 완료** |
| 시간 | 라운드 후 수일 소요 | **예약 시 즉시 정산** |
| 영수증 | 대표 결제자만 보유 | **각자 개별 영수증** |

## 4. 수익 모델

### 4.1 1단계: 간편결제 수수료 (서비스 런칭 시)

간편결제(카드결제/더치페이) 이용 시 **결제 금액의 5%** 수수료 부과. 현장결제는 수수료 없음.

#### 수수료 정책

| 항목 | 내용 |
|------|------|
| **수수료율** | 결제 금액의 **5%** (100원 단위 올림) |
| **최소 수수료** | **300원** (5% 계산 결과가 300원 미만이면 300원 적용) |
| **현장결제** | **수수료 없음** — 진입 장벽 제거, 가맹점 유치 |
| **적용 대상** | 간편결제 (카드결제, 더치페이) |
| **더치페이** | 참여자별 인당 결제 금액 기준으로 각각 수수료 부과 |
| **포함 항목** | Toss 카드결제 수수료(2.8%) + 플랫폼 이용료 |
| **가맹점 부담** | 없음 (수수료는 이용자 부담) |

#### 수수료 계산 방식

```
수수료 = max(올림100(결제금액 × 5%), 300원)

올림100: 100원 단위 올림 (예: 250원 → 300원, 500원 → 500원)
최소 수수료: 300원
```

#### 인당 이용료별 수수료표

| 인당 이용료 | 5% 계산 | 100원 올림 | 최소 적용 | **최종 수수료** |
|-----------|--------|----------|---------|--------------|
| 3,000원 | 150원 | 200원 | 최소 300원 | **300원** |
| 4,000원 | 200원 | 200원 | 최소 300원 | **300원** |
| 5,000원 | 250원 | 300원 | = | **300원** |
| 6,000원 | 300원 | 300원 | = | **300원** |
| 7,000원 | 350원 | 400원 | - | **400원** |
| 8,000원 | 400원 | 400원 | - | **400원** |
| 10,000원 | 500원 | 500원 | - | **500원** |
| 15,000원 | 750원 | 800원 | - | **800원** |
| 20,000원 | 1,000원 | 1,000원 | - | **1,000원** |

#### 수수료 예시 (실제 시나리오)

| 시나리오 | 이용료 | 결제 방식 | 수수료 | 결제 총액 |
|---------|--------|----------|--------|----------|
| 1인 카드결제 | 10,000원 | 간편결제 | 500원 | **10,500원** |
| 1인 카드결제 | 15,000원 | 간편결제 | 800원 | **15,800원** |
| 2인 더치페이 | 인당 7,000원 | 더치페이 | 인당 400원 | 인당 7,400원 **(총 14,800원)** |
| 4인 더치페이 | 인당 5,000원 | 더치페이 | 인당 300원 | 인당 5,300원 **(총 21,200원)** |
| 4인 더치페이 | 인당 10,000원 | 더치페이 | 인당 500원 | 인당 10,500원 **(총 42,000원)** |
| 1인 현장결제 | 10,000원 | 현장결제 | **0원** | **10,000원** |

#### 플랫폼 수익 구조 (Toss 수수료 차감 후)

| 인당 이용료 | 수수료 | Toss 2.8% | **플랫폼 순수익** |
|-----------|--------|-----------|----------------|
| 5,000원 | 300원 | 140원 | **160원** |
| 7,000원 | 400원 | 196원 | **204원** |
| 10,000원 | 500원 | 280원 | **220원** |
| 15,000원 | 800원 | 420원 | **380원** |

> 낮은 수수료(5%)로 간편결제 전환율을 높이고, 더치페이 활성화로 결제 건수를 극대화하는 전략

#### 간편결제 전환 유도 전략

| 전략 | 내용 |
|------|------|
| 현장결제 무료 유지 | 수수료 부담 Zero → 진입 장벽 완전 제거, 가맹점 유치 |
| 투명한 수수료 | 5% 단일 요율 → 시니어도 직관적으로 이해, 신뢰 확보 |
| 간편결제 혜택 | 예약 확정 보장, 노쇼 시 자동 환불 처리, 이용 기록 관리 |
| 더치페이 편의성 | "각자 결제" 원클릭 → 모임 정산 고민 해결 (핵심 전환 동력) |
| 결제 내역 관리 | 간편결제 시 앱 내 영수증, 이용 기록 자동 관리 |

#### 1단계 수익 전망

> 더치페이 시 인당 개별 결제이므로, 예약 1건에 결제 건수가 배수로 증가

| 시나리오 | 월 예약 건수 | 간편결제율 | 월 결제 건수 | 평균 수수료 | **월 수익** | **연 수익** |
|---------|-----------|----------|-----------|-----------|-----------|-----------|
| 초기 (6개월) | 5,000건 | 35% | 5,000건 | 400원 | **200만** | **2,400만** |
| 성장기 (1년) | 15,000건 | 50% | 20,000건 | 420원 | **840만** | **1.01억** |
| 안정기 (2년) | 40,000건 | 65% | 60,000건 | 440원 | **2,640만** | **3.17억** |
| 확장기 (3년) | 80,000건 | 75% | 120,000건 | 450원 | **5,400만** | **6.48억** |

> 결제 건수 = 간편결제 예약 × 평균 결제 인원 (1인 결제 + 더치페이 배수 효과)

### 4.2 2단계: 레슨 마켓플레이스 "메이트 레슨" (런칭 1년 후)

파크골프 전문 레슨 중개 플랫폼. 인플루언서 생태계 모델을 적용한 **데이터 기반 프로 성장 시스템**.

#### 시장 기회

| 비교 | 일반 골프 레슨 | 파크골프 레슨 |
|------|-------------|-------------|
| 시장 상태 | 레드오션 (수많은 플랫폼) | **블루오션** (전문 플랫폼 전무) |
| 레슨 가격 | 1회 10~30만원 | 1회 **2~6만원** (진입 장벽 낮음) |
| 이용자 특성 | 20~50대 | **50~70대** (건강 + 사교 목적) |
| 반복 수요 | 실력 향상 위주 | **사교 + 건강 + 실력** (재구매율 높음) |
| 그룹 레슨 비율 | 30% | **70%** (동호회 기반) |

#### 프로 티어 시스템 — 인플루언서 성장 모델

기존 레슨 플랫폼이 자격증 기반 정적 인증이라면, 메이트 레슨은 **데이터 기반 동적 티어 시스템**입니다. 인플루언서 생태계에서 Nano → Micro → Macro → Mega로 성장하듯, 레슨 프로도 실적 데이터로 성장합니다.

```
┌─────────────────────────────────────────────────────────┐
│  마스터 프로  ★★★★  (Big Influencer)                     │
│  플랫폼 대표 강사, 콘텐츠 제작, 브랜드 앰배서더            │
├─────────────────────────────────────────────────────────┤
│  스타 프로    ★★★   (Macro Influencer)                   │
│  지역 대표 강사, 그룹레슨 전문, 동호회 출장                │
├─────────────────────────────────────────────────────────┤
│  인기 프로    ★★    (Micro Influencer) ← 핵심 수익 계층   │
│  충성 수강생 보유, 높은 재수강률, 실력 향상 입증            │
├─────────────────────────────────────────────────────────┤
│  신규 프로    ★     (Nano Influencer)                     │
│  자유 가입, 프로필 작성, 첫 레슨 시작                      │
└─────────────────────────────────────────────────────────┘
```

> **마이크로 프로(★★ 인기)가 플랫폼의 실질 매출을 만든다** — 인플루언서 마케팅에서 마이크로(1K~10K)의 참여율이 메가(100K+)보다 3~5배 높은 것처럼, 소수 충성 수강생을 가진 인기 프로의 재수강률이 가장 높다.

#### 프로 평가 지표 (프로 스코어카드)

| 인플루언서 지표 | 레슨 프로 지표 | 측정 방법 | 가중치 |
|--------------|-------------|----------|--------|
| 팔로워 수 | 누적 수강생 수 | 레슨 완료 건수 | 15% |
| **참여율 (Engagement)** | **재수강률** | 재수강 비율 (핵심 지표) | **30%** |
| 도달률 (Reach) | 신규 수강생 유입 | 월별 신규 수강생 수 | 10% |
| **콘텐츠 품질** | **레슨 평점** | 수강생 리뷰 평균 (5점 만점) | **25%** |
| **전환율** | **수강생 실력 향상도** | 레슨 전후 스코어 변화 | **20%** |

> 재수강률(30%) + 평점(25%) + 실력 향상도(20%) = **품질 지표 75%** — "많이 가르친 사람"이 아니라 **"잘 가르친 사람"**이 올라가는 구조

#### 티어별 승격 기준과 혜택

| 티어 | 승격 기준 | 검색 노출 | 수수료 | 플랫폼 지원 |
|------|---------|----------|--------|-----------|
| **신규 ★** | 자유 가입 | 기본 | 15% | - |
| **인기 ★★** | 레슨 30건+, 평점 4.0+, 재수강률 40%+ | 우선 노출 | 13% | 프로필 배지 |
| **스타 ★★★** | 레슨 100건+, 평점 4.5+, 재수강률 55%+ | 상위 노출 + AI 추천 | 11% | 프로모션 지원, 그룹레슨 매칭 |
| **마스터 ★★★★** | 레슨 300건+, 평점 4.7+, 재수강률 65%+ | 최상위 + 메인 피처 | 9% | 콘텐츠 제작 지원, 앰배서더 |

> 티어가 올라갈수록 **수수료 인하** (15% → 9%) — 프로의 성장 인센티브
> 승격은 매월 자동 평가, 강등은 분기 단위 (급격한 변동 방지)

#### 프로 성장 플로우

```
신규 프로 가입 (자유 등록)
  ├─ 프로필 작성 (경력, 전문 분야, 레슨 스타일)
  ├─ 자격증 업로드 (선택, 있으면 "자격 인증" 배지)
  ▼
첫 레슨 시작 (AI가 근처 수강생과 매칭)
  ├─ 수강생 리뷰 축적
  ├─ 재수강 발생 → 재수강률 상승
  ├─ 수강생 스코어 개선 → 실력 향상도 반영
  ▼
데이터 기반 자동 평가 (매월 갱신)
  ├─ 기준 충족 → 자동 승격 알림 (★ → ★★)
  ├─ 기준 미달 → 유지
  ▼
인기 프로 ★★ (Micro Influencer 존)
  ├─ 충성 수강생 5~15명, 높은 재수강률
  ├─ "나만의 수강생"이 입소문 → 자연 성장
  ▼
스타 프로 ★★★ (지역 대표)
  ├─ 동호회 출장 레슨, 그룹 레슨 매칭
  ├─ 플랫폼 추천 프로로 상위 노출
  ▼
마스터 프로 ★★★★ (플랫폼 앰배서더)
  ├─ 레슨 팁 콘텐츠 제작 (앱 내 게시)
  ├─ 신규 프로 멘토링
  └─ 플랫폼 브랜드 대표
```

#### 기존 레슨 플랫폼과의 차별화

| 기존 레슨 플랫폼 | 메이트 레슨 (인플루언서 모델) |
|----------------|--------------------------|
| 자격증 있으면 인증, 없으면 비인증 | **데이터 기반 4-Tier 동적 평가** |
| 한번 인증하면 영구 유지 | **매월 성과 기반 자동 갱신** |
| 강사 수(공급량)로 경쟁 | **강사 품질(재수강률·실력향상)**로 경쟁 |
| 가격으로 차별화 | **프로 스코어카드**로 차별화 |
| 플랫폼은 단순 중개 | **플랫폼이 성장 경로와 인센티브 제공** |

#### 레슨 상품 구성

| 레슨 타입 | 인원 | 시간 | 가격대 | 수수료 (9~15%) |
|----------|------|------|--------|--------------|
| **1:1 개인 레슨** | 1명 | 60분 | 4~6만원 | 3,600~9,000원 |
| **그룹 레슨** | 2~4명 | 90분 | 인당 2~3만원 | 인당 1,800~4,500원 |
| **초보 입문반** | 4~8명 | 120분 × 4회 | 인당 8~12만원 (패키지) | 인당 7,200~18,000원 |
| **코스 공략** | 2~4명 | 현장 라운드 | 인당 3~5만원 | 인당 2,700~7,500원 |
| **동호회 출장 레슨** | 8~16명 | 180분 | 총 20~40만원 | 18,000~60,000원 |

> 수수료 범위는 프로 티어에 따라 결정 (신규 15% ~ 마스터 9%)

#### AI 연계 기능

| 기능 | 설명 |
|------|------|
| **AI 프로 매칭** | "스윙 교정 배우고 싶어" → 프로 스코어카드 + 전문 분야 + 위치 + 가격 기반 최적 추천 |
| **레슨 + 예약 통합** | 레슨 예약 시 골프장 슬롯 동시 확보 (프로 스케줄 + 빈 시간 자동 매칭) |
| **AI 실력 분석** | 라운드 기록 기반 약점 분석 → "퍼팅 정확도가 낮아요, 퍼팅 전문 프로 추천" |
| **레슨 효과 추적** | 레슨 전후 스코어 비교 → 효과 시각화, 프로 스코어카드에 반영 |
| **프로 성장 대시보드** | 프로 전용 화면: 스코어카드, 수강생 현황, 티어 진행률, 수익 통계 |

#### 레슨 예약 플로우

```
사용자: "퍼팅 레슨 받고 싶어"

→ AI: 라운드 기록 분석 → 약점 파악 (퍼팅 정확도 62%)
→ AI: 근처 퍼팅 전문 프로 추천 (프로 스코어카드 기반)
       ├─ 김프로 ★★★ (재수강률 62%, 평점 4.8, 1회 5만원)
       ├─ 이프로 ★★  (재수강률 55%, 평점 4.6, 1회 4만원)
       └─ 박프로 ★★★ (재수강률 58%, 평점 4.9, 그룹 인당 2.5만원)
→ 사용자: "박프로 그룹 레슨, 이번 주 토요일"
→ AI: 프로 스케줄 + 골프장 빈 시간 매칭 → 예약 확정
→ AI: 동호회 멤버에게 그룹 레슨 참여 요청 브로드캐스트
```
