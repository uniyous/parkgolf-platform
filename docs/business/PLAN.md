## 1. 사업 개요

### 1.1 비전

> "말 한마디로 예약 완료" — 대한민국 최초 AI 파크골프 통합 플랫폼

### 1.2 사업 방향

| 단계 | 목표 | 핵심 전략 |
|------|------|----------|
| **현재** | 초기 고객 확보 | AI 부킹으로 편의성 극대화, 무료 가맹점 유치 |
| **1단계** | 수익 기반 구축 | 예약 수수료 모델 도입 |
| **2단계** | 수익 확대 | 프리미엄 구독 서비스 "메이트 PRO" |
| **3단계** | 사업 확장 | 레슨 마켓플레이스 "메이트 레슨" |

### 1.3 사업 구성

```
파크골프메이트
├── 사용자 앱 (iOS / Android / Web)
│   ├── AI 자연어 예약
│   ├── 그룹 채팅방 단체 예약
│   ├── 더치페이 분할결제
│   └── 소셜 (친구, 채팅, 모임)
│
├── 가맹점 관리 대시보드 (Web)
│   ├── 예약/슬롯 관리
│   ├── 매출 분석
│   ├── 고객/노쇼 관리
│   └── 정책 설정 (취소/환불/노쇼/운영)
│
└── 플랫폼 관리 대시보드 (Web)
    ├── 전국 가맹점 통합 관리
    ├── 통합 통계 및 분석
    ├── 계층형 정책 관리 (PLATFORM → COMPANY → CLUB)
    └── 관리자/역할/권한 관리 (RBAC)
```

## 2. 시장 현황과 기회

### 2.1 파크골프 시장 규모

| 지표 | 수치 | 비고 |
|------|------|------|
| 이용자 수 | **150만+** | 2024년 기준 |
| 전국 파크골프장 | **490+** | 운영 중 |
| 10년 성장률 | **853%** | 2013~2023 |
| 연간 성장률 | **30%+** | 최근 5년 평균 |

### 2.2 이용자 특성

| 항목 | 내용 |
|------|------|
| 핵심 이용층 | 50~70대 시니어 |
| 성별 비율 | 남성 71.5%, 여성 28.5% (여성 참여율 증가 추세) |
| 이용 동기 | 건강 관리, 사교 활동, 여가 |
| 이용 빈도 | 주 2~3회 (동호회 기반 정기 라운드) |
| 1회 이용료 | 5,000~15,000원 |

### 2.3 시장의 문제점

| 문제 | 현황 | 영향 |
|------|------|------|
| **디지털 인프라 부재** | 150만 이용자가 전화/수기 예약 | 시니어 불편, 운영 비효율 |
| **전화 예약** | 6단계, 3~5분 소요, 영업시간만 가능 | 예약 포기, 노쇼 관리 불가 |
| **기존 앱** | 9단계 복잡 UI, 시니어 진입장벽 높음 | "그냥 전화가 편해" |
| **레슨 플랫폼 부재** | 파크골프 전문 레슨 매칭 서비스 없음 | 레슨 수요가 비공식 채널에 분산 |
| **가맹점 관리** | 수기 장부, 노쇼 관리 불가, 통계 부재 | 운영 비효율, 매출 분석 불가 |

### 2.4 기회 요약

```
급성장하는 시장 (연 30%+) + 디지털 인프라 전무 + 레슨 플랫폼 부재
= 선점 기회
```

## 3. 서비스 소개

### 3.1 AI 예약 — 핵심 차별화

| 비교 | 전화 예약 | 기존 앱 | 파크골프메이트 |
|------|---------|--------|-------------|
| 예약 단계 | 6단계 | 9단계 | **1단계** |
| 소요 시간 | 3~5분 | 2~3분 | **~15초** |
| 가능 시간 | 영업시간만 | 24시간 | **24시간** |
| 빈 시간 검색 | 불가 | 수동 | **자동** |
| 날씨 확인 | 불가 | 불가 | **자동** |
| 단체 예약 | 개별 연락 | 개별 예약 | **@AI 한번으로** |

**AI 예약 플로우:**

```
사용자: "천안 골프장 2명 예약해 줘"
  → AI: 골프장 검색 + 날씨 확인 + 빈 슬롯 조회
  → 사용자: 슬롯 선택 + 결제방법 선택
  → AI: 예약 완료 (~15초)
```

## 4. 수익 모델

### 4.1 1단계: 예약 수수료 (서비스 런칭 시)

카드결제/더치페이 이용 시 이용료 구간별 정액 수수료 부과. 카드 결제 수수료(Toss 2.8%) 포함.

#### 수수료 체계

| 이용료 구간 | 수수료 (VAT 포함) | 수수료율 환산 |
|------------|-------------------|-------------|
| 5,000원 미만 | **500원** | ~10% |
| 5,000 ~ 9,999원 | **1,000원** | 10~20% |
| 10,000 ~ 19,999원 | **1,500원** | 7.5~15% |
| 20,000원 이상 | **2,000원** | ~10% 이하 |

#### 수수료 정책

| 항목 | 내용 |
|------|------|
| 적용 대상 | 카드결제, 더치페이 (현장결제는 수수료 없음) |
| 더치페이 | 참여자별 각각 수수료 부과 (4명 더치페이 = 수수료 × 4) |
| 포함 항목 | Toss 카드결제 수수료(2.8%) + 플랫폼 이용료 |
| 가맹점 부담 | 없음 (수수료는 이용자 부담) |

#### 수수료 예시

| 시나리오 | 이용료 | 결제 방식 | 수수료 | 결제 총액 |
|---------|--------|----------|--------|----------|
| 1인 예약 | 10,000원 | 카드결제 | 1,500원 | 11,500원 |
| 2인 더치페이 | 인당 5,000원 | 더치페이 | 인당 1,000원 | 인당 6,000원 (총 12,000원) |
| 4인 더치페이 | 인당 3,750원 | 더치페이 | 인당 500원 | 인당 4,250원 (총 17,000원) |
| 1인 예약 | 20,000원 | 카드결제 | 2,000원 | 22,000원 |

#### 카드결제 전환 유도 전략

| 전략 | 내용 |
|------|------|
| 현장결제 유지 | 수수료 없음 → 진입 장벽 제거, 가맹점 유치 |
| 카드결제 혜택 | 예약 확정 보장, 노쇼 시 자동 환불 처리 |
| 더치페이 편의성 | "각자 결제" 원클릭 → 모임 정산 고민 해결 |
| 결제 내역 관리 | 카드결제 시 앱 내 영수증, 이용 기록 자동 관리 |

#### 1단계 수익 전망

| 시나리오 | 월 카드결제 건수 | 평균 수수료 | 월 수익 | 연 수익 |
|---------|----------------|-----------|--------|--------|
| 초기 (6개월) | 3,000건 | 1,200원 | 360만원 | 4,320만원 |
| 성장기 (1년) | 10,000건 | 1,300원 | 1,300만원 | 1.56억원 |
| 안정기 (2년) | 30,000건 | 1,400원 | 4,200만원 | 5.04억원 |

### 4.2 2단계: 프리미엄 구독 "메이트 PRO" (런칭 6개월 후)

AI 기능을 대폭 강화한 프리미엄 구독 서비스.

#### 가격

| 플랜 | 가격 | 비고 |
|------|------|------|
| 월간 구독 | **9,900원/월** | |
| 연간 구독 | **99,000원/년** | 월 8,250원, 2개월 무료 |

#### 무료 vs PRO 비교

| 카테고리 | 기능 | 무료 회원 | 메이트 PRO |
|---------|------|---------|-----------|
| **AI 예약** | 기본 자연어 예약 | O | O |
| **AI 자동 예약** | 조건 설정 → AI가 매주 자동 예약 | - | O |
| **AI 빈자리 알림** | 만석 시 취소 발생 즉시 알림 + 자동 예약 | - | O |
| **AI 코스 추천** | 라운드 기록 분석 → 실력 맞춤 추천 | - | O |
| **AI 날씨 매칭** | 주간 날씨 분석 → 최적 라운드 날짜 추천 | 당일만 | O |
| **AI 모임 관리** | 정기 모임 자동 스케줄링, 참석률 분석 | - | O |
| **우선 예약** | 인기 슬롯 조기 오픈 | 일반 | **30분 먼저** |
| **노쇼 면제** | 월 노쇼 페널티 면제 | 0회 | **월 1회** |
| **취소 수수료** | 취소 시 수수료 할인 | 정책 기준 | **50% 할인** |
| **라운드 기록** | 스코어 기록 및 통계 | 최근 10회 | **무제한 + 트렌드 분석** |
| **광고** | 앱 내 광고 | 있음 | **없음** |

#### 킬러 기능 상세

**AI 자동 예약:**

```
설정: "매주 토요일, 천안 근처, 날씨 맑음, 오전 10~11시, 2명"

→ AI가 매주 수요일에 조건 확인
  ├─ 날씨 맑음 → 자동 예약 + push 알림
  │   "이번 토요일 10:30 천안시민파크골프장 예약했어요! (맑음 18°C)"
  │
  └─ 날씨 나쁨 → 보류 + push 알림
      "이번 주 토요일은 비 예보라 보류했어요. 대신 일요일은 맑아요, 예약할까요?"
```

**AI 빈자리 알림:**

```
요청: "대전 유성 파크골프장, 이번 주 토요일 오전" (만석)

→ AI가 해당 슬롯 취소 발생 모니터링
  ├─ 취소 발생 → 즉시 push 알림
  │   "대전 유성 토요일 10:00 빈자리 생겼어요!"
  │
  └─ 자동 예약 ON 시 → 즉시 예약 확보
      "대전 유성 토요일 10:00 자동 예약 완료!"
```

**AI 모임 관리:**

```
설정: "골프 동호회, 매월 첫째/셋째 토요일, 8~12명"

→ AI가 자동으로:
  ├─ 멤버에게 참석 확인 요청 (push + 채팅방)
  ├─ 참석 인원 기반 팀 자동 편성 (4명 × N팀)
  ├─ 날씨 좋은 골프장 추천
  ├─ 팀별 슬롯 자동 예약
  └─ 참석률 통계 리포트 (월별)
```

### 4.3 3단계: 레슨 마켓플레이스 "메이트 레슨" (런칭 1년 후)

파크골프 전문 레슨 중개 플랫폼. **플랫폼 수수료 15%**.

#### 시장 기회

| 비교 | 일반 골프 레슨 | 파크골프 레슨 |
|------|-------------|-------------|
| 시장 상태 | 레드오션 (수많은 플랫폼) | **블루오션** (전문 플랫폼 전무) |
| 레슨 가격 | 1회 10~30만원 | 1회 **2~6만원** (진입 장벽 낮음) |
| 이용자 특성 | 20~50대 | **50~70대** (건강 + 사교 목적) |
| 반복 수요 | 실력 향상 위주 | **사교 + 건강 + 실력** (재구매율 높음) |
| 그룹 레슨 비율 | 30% | **70%** (동호회 기반) |

#### 레슨 프로 등록

| 항목 | 내용 |
|------|------|
| 자격 요건 | 파크골프 협회 자격증, 경력 3년 이상, 또는 핸디캡 기준 |
| 프로필 | 경력, 전문 분야, 레슨 스타일, 수강생 후기, 평점 |
| 인증 | 자격증 사본 업로드 → 플랫폼 심사 후 인증 배지 부여 |
| 정산 | 월 2회 자동 정산 (수수료 15% 차감 후 입금) |

#### 레슨 상품 구성

| 레슨 타입 | 인원 | 시간 | 가격대 | 수수료 (15%) |
|----------|------|------|--------|-------------|
| **1:1 개인 레슨** | 1명 | 60분 | 4~6만원 | 6,000~9,000원 |
| **그룹 레슨** | 2~4명 | 90분 | 인당 2~3만원 | 인당 3,000~4,500원 |
| **초보 입문반** | 4~8명 | 120분 × 4회 | 인당 8~12만원 (패키지) | 인당 12,000~18,000원 |
| **코스 공략** | 2~4명 | 현장 라운드 | 인당 3~5만원 | 인당 4,500~7,500원 |
| **동호회 출장 레슨** | 8~16명 | 180분 | 총 20~40만원 | 30,000~60,000원 |

#### AI 연계 기능

| 기능 | 설명 |
|------|------|
| **AI 프로 매칭** | "스윙 교정 배우고 싶어" → 전문 분야, 위치, 시간, 가격 기반 최적 프로 추천 |
| **레슨 + 예약 통합** | 레슨 예약 시 골프장 슬롯 동시 확보 (프로 스케줄 + 빈 시간 자동 매칭) |
| **AI 실력 분석** | 라운드 기록 기반 약점 분석 → "퍼팅 정확도가 낮아요, 퍼팅 전문 프로 추천" |
| **레슨 효과 추적** | 레슨 전후 스코어 비교 → 효과 시각화, 프로 평가 근거 |

#### 레슨 예약 플로우

```
사용자: "퍼팅 레슨 받고 싶어"

→ AI: 라운드 기록 분석 → 약점 파악 (퍼팅 정확도 62%)
→ AI: 근처 퍼팅 전문 프로 3명 추천
       ├─ 김프로 (평점 4.8, 1회 5만원, 토/일 가능)
       ├─ 이프로 (평점 4.6, 1회 4만원, 평일 가능)
       └─ 박프로 (평점 4.9, 그룹 인당 2.5만원, 토 가능)
→ 사용자: "박프로 그룹 레슨, 이번 주 토요일"
→ AI: 프로 스케줄 + 골프장 빈 시간 매칭 → 예약 확정
→ AI: 동호회 멤버에게 그룹 레슨 참여 요청 브로드캐스트
```
